# Brainpost Video

## A FFMPEG Ruby interface to create feed and stories videos

### Setup
```
FFMPEG >= 4.3
Ruby >= 2.3
```

```shell
# on linux
sudo get install -y build-essential libxi-dev libglu1-mesa-dev libglew-dev pkg-config
```


### Run exemples
```shell
# create temporary folder
mkdir tmp 
```

#### Feed with logo
```
  ruby examples/feed.rb
```
